EEIS Prayer Time App Backup
Created on: 2026-02-06

To restore this project on a new PC:
1. Copy this folder to your local drive (e.g., C:\Projects\eeis-app).
2. Open a terminal in that folder.
3. Run `npm install` to download all dependencies (this recreates the node_modules folder).
4. Run `npm run dev` to start the development server.
